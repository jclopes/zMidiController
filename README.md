# MidiConsoleApplication
I started this project because I wanted to trigger midi events while live recording on ableton live.
I had the option to spend $$$ money on a launchpad midi controller or use the multiple console controllers that I already owned.
After searching the web, there was no simple application that could translate joystick/gamepad events to midi events. (There are some plugins for max4live but you need to own max4live which cost money $$$ and that's what we don't want.)
This sounded like a perfect project to learn my self some modern C/C++.
